# 📱 OFFLINE MCQ QUIZ APP

A modern Android quiz application built using **Jetpack Compose** and **Room Database**.
This app allows users to take quizzes offline with a smooth UI, timer-based questions, and instant results.

---

## 🚀 Features

* 🧠 Multiple Choice Questions (MCQ)
* ⏱️ Countdown Timer for each question
* 📊 Score calculation and result screen
* 💾 Offline storage using Room Database
* 🎨 Modern UI with Jetpack Compose
* 🔄 Restart Quiz functionality
* ❌ Exit button to close the app

---

## 🛠️ Tech Stack

* **Kotlin**
* **Jetpack Compose**
* **Room Database**
* **MVVM Architecture**
* **Coroutines & Flow**

## ⚙️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/mayank-gulshan/OFFLINE_MCQ_QUIZ.git
   ```

2. Open in **Android Studio**

3. Sync Gradle

4. Run the app on:

   * Emulator OR
   * Real Android device
